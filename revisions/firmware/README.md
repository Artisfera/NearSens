# NearSens firmware archive

This directory contains the NearSens firmware versions that were found, archived and preserved. They were collected from working copies, archives and older saved project variants.

There may be gaps in the numbering. During development, some versions were overwritten or were not saved as separate packages. Some versions also have several different variants, placed in separate subfolders.

This is a working archive. Directory names such as final, final-safe or final-tuned come from testing stages and do not mark the current project release. The current firmware is in `firmware/NearSens_1.0.0`.
